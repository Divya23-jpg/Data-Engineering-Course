AWS S3
Amazon Web Service simple  storage service (S3)
    1.It is  a scalable object storage service used to store and retrive data from abywhere on the internet.
    2.It stores data as an object inside a bucket and supports virtually unlimited storage capacity. with high durability,
    security and Availaibality.
    3.Amazon S3 also provides multiple storage classes to optimise cost based on data acces pattern.


Core Architecture/ Function availabile 

    1.S3 Utilizer a flat, non-hierarical Architecture that allows servixe to scale virtually.
      Scale virtually infinitely.

2. The fundamental building blocks are logical continious and the data units stored within them.
        
    1.Bucket: A container for objects which has the following characteristics
       1. Global namespace: Every bucket names must be unique value across all the AWS Account worldwide(Like DNA name).

       2.Region spacific: You create a bucket in a specific region (US-east-1) and your data 
       never leaves that region unless you explicitely move it.

    2.Objects: The fundamental entity stored in S3 , consisting of :
        1.Key: The name of the object [image.jpg]
        2.Value: The data itsel [bytes]
        3.Metadata: name-value pairs describes the object [Content-type : image/jpg]
        4.size: Objects can range from 0 bytes  upto 5 Terabyte
