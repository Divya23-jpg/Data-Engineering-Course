Regions: Physical locations worldwide where AWS clusters data center

             ap-south-1(Mumbia)

Availaibality Zones (AZ):
    Multiple isolated location within each region
    Each AZ contains one or more descrete data centers.
    Ensures High Availaibality(HA) and disaster recovery(DA).
    Prevents single point of failure.



Edge location: 
    Smaller data center in major cities worldwide.
    Used to cache content closer to the end user with the help of AWS cloud fund(CDN)
