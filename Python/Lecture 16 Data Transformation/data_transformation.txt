Data Transformation

1.Type Casting: Change the type of the column 
    Ex: Float into Int
    df['price']=df['price'].astype('float')

2.Mapping: Replaces the value based on the Dictinory or another seies
    Ex: df['Status_code']=df['Status'].map({"Y":1,"N":0})

3.Applying Function: Executes a custom Python function along the axis (rows or column) of a DataFrame
    Method: .apply()
    EX: df['tax']=df.apply(lambda row: row['name']* 0.15 if row[])

4.Group-Wise Transformation:  Applies an aggregate function (like .mean() ) to group returns a result with same shape as original DataFrame
    Method: .transform 
    EX: df['avg_dep_salary']=df.groupby('dept')['salary'].transform('mean')



Relational Logic:
    Pandas have few Methods to apply joins,concatenation,mapping,slicing

    1.Mapping: Combine DataFrame on shared keys (column or indexes)
    Method: .merge() 
    EX: df_merged=df1.merge(df2,on='employe_id',have='inner Join')

    2.concatenation: Stacks DataFrame vertically by rows or horizontally by column
    Method: .concat()
    EX: df_stacked=pd.concat([df1,df2],axis=0)

    3.Filtering & Logic: Two Dimensional Data
    Method: .loc[] : Checking based on Labels
            .iloc[]:Checking based on Indexing
