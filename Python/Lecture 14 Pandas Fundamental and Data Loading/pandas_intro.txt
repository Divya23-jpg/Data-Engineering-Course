Pandas is a powerfull , open source Python Library designed for 
data manipulation,Analysis and cleaning

Core Data structure:
    1.Series:
        A one dimentional labeled array capable of holding 
        any datatype (int,string etc)

    2.Data Frame:
        A two dimentional size ,mutable, and potentially hetrogenous
        tabular data structure with the labeles axes(rows and cols)


Pandas is used for :
    1.Data Cleaning
    2.Data manipulation
    3.Data Transformation
    4.Data Imputation


Essential Features:

1.Data Input/Output: 
    Reading from and writing to format like CSV,Excel,SQL databases,and Json,

2.Cleaning:
    Builtin function to handle ,remove,fill missing data (NAN) and duplicate

3.Transformation:
    Filtering , Merging, Reshaping, Creating Pivoting (rows to col, cols to rows) large dataset

4. Aggregation:
    Using "Group By" operations to organize data into categories and compute statistics like Sum or Average


5.Time series:
    sophesticated functionality for data  based on the date includinh the frequenecy coversion
